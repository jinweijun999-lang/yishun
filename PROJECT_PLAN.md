## 一、产品定位

### 目标用户
- **核心用户**：海外华人（北美、东南亚、欧洲），25-50岁，对命理感兴趣
- **用户画像**：
  - 海外华人留学生/新移民：面临人生重大决策，寻求文化认同感
  - 华裔商务人士：事业有成但命理认知需求未被满足
  - 命理爱好者：需要工具辅助学习

### 核心价值主张
> **"让命理成为你的人生指南，而不是迷信"**

## 二、变现模式

| 模式 | 说明 | 预期占比 | 优先级 |
|------|------|----------|--------|
| **Freemium** | 四柱排盘基础版免费，高级功能付费 | 导流 | ⭐⭐⭐ |
| **报告购买** | 单次深度报告（流年、姻缘、事业）$3-15/份 | 主要 | ⭐⭐⭐ |
| **会员订阅** | 月/年订阅解锁全部功能，$9.9/月或 $89/年 | 核心 | ⭐⭐⭐ |

| **高级课程** | 命理系统课程 | 增值 | ⭐ |
| **看广告解锁** | 看完广告可免费领取结果 | 获客 | ⭐⭐⭐ |

### 变现补充：看广告解锁
- **机制**：用户可选择看完整广告（15-30秒）来免费解锁分析结果
- **限制**：每个结果限用1次/天，防止滥用
- **广告SDK**：Google AdMob / Meta Audience Network
- **触发场景**：
  - 免费用户查看高级分析结果前
  - 解锁完整报告时
  - 获取更多排盘次数时

### 定价策略
- **心理价位**：海外华人付费意愿高于国内，参考 Calm/Astrolync
- **首单优惠**：新用户首单 50% off
- **家庭计划**：支持家庭成员绑定

## 三、推广方案

### 3.1 社媒营销（核心渠道）
- **小红书海外版**：内容种草（命理故事、每日星座运势）
- **Instagram/TikTok**：短视频"今日运势"病毒内容
- **YouTube**：命理知识科普（SEO 长尾流量）
- **Facebook 华人社群**：精准投放

### 3.2 KOL 合作
- 命理类 KOL：YouTube/TikTok 命理博主，定制内容+联盟链接
- 文化类 KOL：海外华人生活博主，场景植入
- **预算分配**：KOL 投放占推广预算 40%

### 3.3 ASO 优化
- **关键词**：Fortune, Astrology, Chinese Zodiac, Bazi
- **本地化**：应用商店描述多语言（英/繁中/马来/越南）

### 3.4 付费广告
- **Google Ads**：搜索广告（"Chinese astrology app"）
- **Meta Ads**：兴趣定向（华人、命理）
- **TikTok Ads**：短视频素材，年轻用户
- **预算建议**：初期 $5000-10000/月测试

## 四、竞品分析

| App | 优势 | 劣势 |
|-----|------|------|
| **Astrolync** | 国际化程度高，UI精美 | 无中文语境，非真正命理 |
| **Fortune Teller** | 用户基数大 | 功能浅，广告多 |
| **Zodiac Master** | 亚洲市场积累 | 设计老旧，无海外本地化 |
| **Qiyao 七曜** | 专业四柱功能 | 仅中文，国内为主 |

### YiShun 差异化
1. **真正全球化**：文化本地化（节日、活动贴合海外华人场景）
2. **专业深度**：四柱+五行+十神一条龙
3. **看广告免费解锁**：低成本获客，用户粘性高
4. **社区氛围**：有温度的命理社区

## 五、技术架构

### 整体架构
```
客户端层: Flutter App (iOS/Android) + React Native (Web)
     ↓ HTTPS
API网关层: Kong / AWS API Gateway
     ↓
业务逻辑层: FastAPI (Python 3.11+)
     ↓
数据层: PostgreSQL + Redis + S3/GCS
```

### 技术选型
| 层级 | 技术 | 理由 |
|-----|------|------|
| 前端移动端 | Flutter | 原生性能，一致UI |
| 前端Web | Next.js | SSR优化，SEO友好 |
| 后端框架 | FastAPI | 异步高性能，文档自动 |
| AI | Gemini 2.0 Flash | 性价比高，支持多模态 |
| 数据库 | PostgreSQL | 关系型，JSON支持 |
| 缓存 | Redis | 高性能 |
| 支付 | Stripe | 海外生态成熟 |

### 开发计划
| 阶段 | 时间 | 任务 |
|------|------|------|
| Phase 1 | 第1-2周 | 项目脚手架、认证系统、数据库 |
| Phase 2 | 第3-5周 | 四柱排盘核心算法、Gemini AI接入 |
| Phase 3 | 第6-7周 | Stripe支付集成、订阅墙逻辑 |
| Phase 4 | 第8周 | 测试、App Store提交 |

## 六、数据统计与埋点

### 埋点设计
| 事件类型 | 事件名称 | 关键属性 |
|---------|---------|---------|
| 用户 | user_register | platform, country |
| 排盘 | chart_created | chart_type, is_premium |
| AI分析 | analysis_generated | analysis_type, tokens_used |
| 付费 | subscription_started | plan_id, payment_method |

### 转化漏斗
```
访问App → 注册账号 → 首次算命 → 付费订阅
  (DAU)   (注册率)    (激活率)    (付费率)
```

### 核心指标
- DAU/MAU/留存率
- CAC < $5（早期），< $10（增长期）
- ARPU > $2/月，LTV > $25
- **广告观看次数**：人均 > 3次/周
- **广告完播率**：> 80%

## 十、风险与建议

### 主要风险
- **命理合规风险**：需做好免责声明
- **获客成本**：海外华人群体小众，CAC相对较高

### 建议
- MVP 优先做 **四柱排盘 + 基础报告**，快速验证市场需求
- 前期聚焦 **北美华人市场**
- 与海外华人社区合作，降低获客成本
